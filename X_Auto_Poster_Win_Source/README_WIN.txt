【X Auto Poster Windows版 ビルド手順】

このフォルダには、Windows上でアプリをビルド（exe化）するために必要なファイルが含まれています。

■ 前提条件
- Python 3.10以上がインストールされていること
  (コマンドプロンプトで `python --version` が動くこと)

■ 手順
1. `setup_and_build.bat` を実行します。

   このスクリプトは以下の処理を自動で行います：
   - Python仮想環境の構築とライブラリインストール
   - Playwright用ブラウザのダウンロード (`dist/browsers`)
   - PyInstallerによるアプリ本体のビルド (`dist/X_Auto_Poster.exe`)
   - **Inno Setupによるインストーラー作成** (Inno Setupがインストールされている場合)

2. 処理が成功すると、`Output` フォルダにインストーラーが生成されます。
   ファイル名: `X_Auto_Poster_Setup_v1.8.2.exe`

■ 配布について
生成された `X_Auto_Poster_Setup_v1.8.2.exe` を配布してください。
ユーザーはこのインストーラーを実行するだけで、アプリとブラウザが正しくインストールされます。

※ もしスクリプトが Inno Setup を見つけられなかった場合は、
   手動で `setup.iss` を Inno Setup で開いてコンパイルしてください。

■ トラブルシューティング
もし「ブラウザが開かない」「アプリがすぐ落ちる」などの問題が発生した場合は、
`debug_run.bat` を実行してください。
黒い画面（コンソール）が表示され、そこにエラー内容が出力されます。
そのエラー内容を開発者に伝えてください。

■ 注意事項
- ビルド中にセキュリティソフトが反応する場合がありますが、誤検知ですので許可してください。
- 初回起動時はブラウザのダウンロードなどで時間がかかる場合があります。
